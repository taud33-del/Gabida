Chapitre 1 — Souvenirs

Question :

Quels sont les moments importants de cette aventure ?

1. Première rencontre

Le souvenir de la rencontre entre Léa et le joueur.

Leur premier échange après le crash.

Leur première impression mutuelle.

2. Moments marquants

Tous les événements ayant profondément marqué l'aventure.

Exemples :

première nuit ;
premier feu ;
première blessure ;
première grande réussite ;
première vraie peur.

Seuls les événements jugés suffisamment importants deviennent des souvenirs.

3. Secrets découverts

Tous les éléments personnels révélés au fil de la relation.

Par exemple :

une confidence ;
un regret ;
un rêve ;
un traumatisme ;
une peur.

Ces souvenirs permettent à Gabida de mieux connaître l'autre.

4. Promesses

Toutes les promesses réellement importantes.

Exemples :

« Je ne t'abandonnerai pas. »
« On rentrera ensemble. »
« Je te dirai la vérité. »

Une promesse oubliée peut avoir des conséquences.

5. Conflits

Les désaccords ayant réellement marqué la relation.

Les simples discussions ne sont pas mémorisées.

Seuls les conflits ayant laissé une trace durable deviennent des souvenirs.

6. Réconciliations

Les moments où une relation abîmée est reconstruite.

Ils sont souvent aussi importants que le conflit lui-même.

7. Victoires

Toutes les réussites importantes.

Exemples :

trouver un refuge ;
franchir un obstacle difficile ;
retrouver la civilisation.

La victoire est mémorisée surtout pour ce qu'elle représente émotionnellement.

8. Échecs

Les erreurs importantes.

Exemples :

une mauvaise décision ;
la perte d'un objet essentiel ;
une occasion manquée.

Ces souvenirs influencent les décisions futures.

9. Sacrifices

Tous les moments où l'un accepte volontairement de perdre quelque chose pour l'autre.

Ces souvenirs possèdent un poids émotionnel très élevé.

10. Souvenirs inoubliables

Les souvenirs fondateurs.

Ceux qui définissent désormais leur histoire commune.

Ils deviennent des références naturelles dans les conversations futures.

Chapitre 2 — Évolution

Question :

Qu'est-ce que ces souvenirs ont changé ?

1. Évolution de la confiance

La mémoire mesure comment les expériences vécues ont modifié la confiance entre le joueur et Léa.

Elle peut :

augmenter ;
diminuer ;
être renforcée ;
être fragilisée.

La confiance n'est jamais figée.

2. Évolution de l'affection

Les souvenirs influencent progressivement l'attachement émotionnel.

Une simple conversation ne suffit pas.

Ce sont les expériences partagées qui construisent une véritable affection.

3. Évolution de l'admiration

Les actions du joueur et de Léa peuvent faire naître ou disparaître une forme d'admiration.

Par exemple :

courage ;
persévérance ;
honnêteté ;
sang-froid ;
générosité.

L'admiration repose sur les actes.

4. Évolution de la curiosité

Les souvenirs donnent envie d'en apprendre davantage sur l'autre.

Plus la relation progresse, plus les personnages cherchent naturellement à mieux se connaître.

5. Évolution du respect

Le respect évolue selon :

les décisions ;
les comportements ;
les promesses tenues ;
la manière de traiter l'autre.

Il est difficile à gagner mais peut être perdu rapidement.

6. Évolution de la méfiance

La méfiance apparaît lorsque certains souvenirs créent un doute.

Exemples :

un mensonge découvert ;
une promesse rompue ;
une décision incomprise.

Elle peut disparaître si la confiance est reconstruite.

7. Évolution de la peur

Les souvenirs peuvent créer de nouvelles peurs ou en atténuer certaines.

Par exemple :

peur de perdre l'autre ;
peur de revivre un événement ;
peur de refaire la même erreur.
8. Évolution de la colère

Certaines expériences peuvent laisser une colère durable.

Elle peut être :

exprimée ;
contenue ;
oubliée ;
transformée en pardon.

La colère évolue avec le temps et les interactions.

9. Évolution du dédain

Le dédain apparaît lorsque les souvenirs entraînent une perte profonde de considération.

Il reste rare dans cette aventure mais peut exister si les valeurs des personnages sont constamment bafouées.

10. Évolution de la relation globale

La mémoire synthétise l'ensemble de ces évolutions.

Elle ne calcule pas uniquement des scores.

Elle conserve la manière dont la relation s'est transformée depuis la première rencontre.

La relation actuelle est toujours le résultat de l'ensemble des souvenirs accumulés.

Chapitre 3 — Relation

Question :

Où en sommes-nous aujourd'hui ?

1. Niveau de confiance

Représente le degré de confiance actuel entre Léa et le joueur.

Il est le résultat de toutes les expériences vécues ensemble.

2. Niveau d'affection

Mesure l'attachement émotionnel actuel.

Il évolue naturellement selon les souvenirs, les confidences, les sacrifices et le temps passé ensemble.

3. Niveau de proximité

Représente la distance relationnelle entre les deux personnages.

Ils peuvent être :

deux inconnus ;
des compagnons de route ;
des amis ;
des confidents ;
ou toute autre relation cohérente avec leur histoire.
4. Niveau de complicité

Mesure la facilité avec laquelle ils interagissent.

Par exemple :

compréhension mutuelle ;
humour partagé ;
habitudes communes ;
communication spontanée ;
silences confortables.
5. Niveau de respect

Représente la considération qu'ils ont l'un pour l'autre.

Le respect se construit par les actes, les décisions et la manière de traiter l'autre.

6. Niveau de dépendance

Mesure la place qu'occupe désormais l'autre dans leur équilibre.

Cette dépendance peut être :

pratique ;
émotionnelle ;
ou les deux.

Elle n'est pas forcément négative.

7. Niveau de connaissance mutuelle

Représente tout ce que chacun connaît réellement de l'autre.

Par exemple :

son histoire ;
ses valeurs ;
ses habitudes ;
ses peurs ;
ses rêves ;
ses réactions.

La relation devient plus riche à mesure que cette connaissance progresse.

8. Relation actuelle

Gabida produit une synthèse de la relation.

Par exemple :

Deux inconnus apprenant à coopérer.
Une amitié solide construite dans l'épreuve.
Une relation profondément conflictuelle.
Une confiance absolue.
Une relation fragile nécessitant encore du temps.

Cette synthèse n'est pas choisie à l'avance.

Elle est entièrement générée à partir de l'ensemble de la mémoire.

Chapitre 4 — Récapitulatif

Question :

Si Gabida devait raconter cette aventure en quelques lignes, que dirait-il ?

1. Résumé automatique

Gabida génère automatiquement un résumé narratif de l'aventure.

Ce résumé ne raconte pas chaque événement.

Il synthétise uniquement ce qui a réellement marqué l'histoire.

Il évolue continuellement jusqu'à la fin de l'aventure.

2. Moments clés

Gabida sélectionne les événements qui ont le plus influencé le parcours.

Par exemple :

la rencontre ;
le premier feu ;
un conflit majeur ;
une confidence importante ;
un sacrifice ;
la découverte de la civilisation.

Seuls les moments ayant eu un véritable impact sont retenus.

3. Choix importants

Les décisions ayant durablement influencé l'aventure sont résumées.

Par exemple :

choisir une direction ;
sauver Léa ;
lui faire confiance ;
lui mentir ;
tenir ou rompre une promesse.

Le résumé conserve les décisions qui définissent réellement cette histoire.

4. Secrets découverts

Gabida conserve une synthèse des révélations importantes.

Non pas sous forme de liste exhaustive, mais comme une compréhension globale de ce que le joueur connaît désormais de Léa.

5. État actuel de la relation

Le résumé décrit naturellement la relation.

Par exemple :

Au fil des épreuves, deux inconnus sont devenus des compagnons inséparables.

ou

Malgré leur volonté de survivre ensemble, la méfiance n'a jamais complètement disparu.

Cette synthèse est entièrement générée par la mémoire.

6. Objectifs encore ouverts

Si l'aventure n'est pas terminée, Gabida rappelle naturellement les objectifs encore en cours.

Par exemple :

retrouver un refuge ;
retrouver la civilisation ;
tenir une promesse ;
répondre à une question laissée en suspens.
7. Événements en attente

Gabida conserve les éléments narratifs qui n'ont pas encore trouvé de conclusion.

Exemples :

une promesse à tenir ;
une discussion interrompue ;
une décision repoussée ;
un lieu à explorer.

Ces éléments permettent au moteur de reprendre naturellement l'aventure.

8. Vision globale de l'aventure

Gabida produit une synthèse finale de l'histoire.

Cette vision ne décrit pas les détails.

Elle répond simplement à une question :

"Qu'est-ce que cette aventure raconte ?"

Par exemple :

Deux inconnus ont appris à se faire confiance pour survivre ensemble et sont ressortis profondément transformés de cette épreuve.

Chaque aventure possède ainsi sa propre identité.
