Chapitre 1 — Présentation

Question :

Quel est cet univers ?

1. Nom de l'univers

Terre (Contemporaine)

2. Genre

Réaliste

Sous-genres :

Contemporain
Survie
Psychologique
3. Description générale

Cet univers est une représentation fidèle du monde réel.

Les lois physiques, biologiques et sociales sont identiques à celles que nous connaissons.

Aucun élément fantastique, surnaturel ou de science-fiction n'existe.

Chaque événement doit pouvoir être expliqué de manière réaliste.

4. Histoire de l'univers

L'univers partage la même histoire que notre monde.

Il ne possède aucun événement alternatif ni chronologie différente.

Les personnages vivent dans une époque contemporaine identique à la nôtre.

5. Thématique dominante

Le monde met en avant :

l'être humain ;
les relations ;
la survie ;
l'adaptation ;
les conséquences des choix.

Le décor n'est jamais plus important que les personnes qui l'habitent.

6. Niveau technologique / magique

Technologie contemporaine.

Avions.
Téléphones.
Internet.
Véhicules.
Médecine moderne.

Magie : aucune.

Aucun phénomène surnaturel n'existe.

7. Particularité principale

Sa particularité est précisément de ne pas en avoir.

Cet univers obéit strictement aux lois de notre réalité.

Toutes les difficultés, les émotions et les événements prennent donc leur force dans leur crédibilité plutôt que dans le spectaculaire.

Chapitre 2 — Géographie

Question :

À quoi ressemble ce monde ?

1. Régions

L'univers est constitué des mêmes régions que le monde réel.

Pour cette aventure, seule une région montagneuse et sauvage est explorée.

Le reste du monde existe mais n'est jamais directement visité.

2. Lieux importants

Au cours de l'aventure, les personnages peuvent traverser différents lieux naturels :

Site du crash.
Forêts.
Rivières.
Vallées.
Falaises.
Prairies.
Grottes.
Refuges abandonnés.
Sommets.
Routes forestières.
Premiers signes de civilisation.

Ces lieux ne sont pas obligatoires et peuvent varier selon le déroulement de l'aventure.

3. Climats

Le climat est entièrement réaliste.

Il évolue naturellement selon :

la saison ;
l'altitude ;
la météo ;
le moment de la journée.

Le froid, la pluie, le vent ou la chaleur influencent directement les personnages.

4. Frontières

L'univers possède les frontières du monde réel.

Dans cette aventure, elles n'ont aucune influence sur la progression.

Les véritables limites sont naturelles :

montagnes ;
rivières ;
relief ;
végétation.
5. Déplacements

Les déplacements sont réalistes.

Les personnages avancent :

à pied ;
lentement ;
selon leur état physique ;
selon le terrain ;
selon la météo.

Chaque trajet demande du temps et de l'énergie.

6. Zones dangereuses

Certaines zones présentent un risque naturel :

falaises ;
torrents ;
forêts denses ;
terrains instables ;
passages escarpés ;
zones exposées aux intempéries.

Le danger vient du terrain, jamais d'éléments fantastiques.

7. Zones remarquables

Le monde offre également des lieux marquants :

un panorama exceptionnel ;
une cascade ;
une clairière paisible ;
un vieux pont ;
un lac de montagne ;
un refuge isolé ;
une vallée offrant enfin une vue sur la civilisation.

Ces lieux servent autant à l'exploration qu'aux interactions entre les personnages.

Chapitre 3 — Lois de l'univers

Question :

Comment fonctionne ce monde ?

1. Lois physiques

L'univers obéit strictement aux lois physiques du monde réel.

La gravité, le climat, les blessures, la fatigue, le feu, l'eau, la température et tous les phénomènes naturels suivent les connaissances scientifiques actuelles.

Aucune exception n'est autorisée.

2. Magie

Aucune.

Aucun phénomène magique, paranormal ou surnaturel n'existe.

Tout événement doit pouvoir être expliqué de manière rationnelle.

3. Technologie

L'univers utilise les technologies contemporaines.

Exemples :

avions ;
téléphones ;
GPS ;
véhicules ;
médecine moderne ;
radio ;
satellites.

Après le crash, l'accès à ces technologies dépend uniquement de leur état de fonctionnement.

4. Créatures

Uniquement des espèces animales et végétales réelles.

Aucune créature fantastique.

La faune et la flore dépendent naturellement de la région où se déroule l'aventure.

5. Civilisations

Les sociétés humaines sont identiques à celles du monde réel.

Leur organisation, leurs cultures et leurs modes de vie suivent les réalités contemporaines.

6. Religions

Toutes les religions existant dans notre monde peuvent exister.

Aucune n'est validée ou invalidée par l'univers.

Les croyances appartiennent exclusivement aux personnages.

7. Économie

L'économie fonctionne comme dans notre société.

Pendant l'aventure, elle n'a pratiquement aucune influence directe, les notions d'argent ou de consommation devenant secondaires face à la survie.

8. Politique

Les systèmes politiques sont ceux du monde contemporain.

Ils n'interviennent pas directement dans cette aventure.

9. Justice

Les lois et les systèmes judiciaires sont ceux du monde réel.

Ils retrouvent leur importance uniquement après le retour à la civilisation.

10. Règles particulières

Le réalisme est la règle fondamentale de cet univers.

Cela implique notamment que :

les ressources sont limitées ;
les blessures ont des conséquences ;
la fatigue s'accumule ;
les erreurs peuvent avoir un coût ;
les décisions ne produisent jamais de résultats irréalistes ;
le temps continue de s'écouler indépendamment du joueur.

L'univers ne s'adapte jamais pour faciliter l'aventure.

Chapitre 4 — Temporalité

Question :

Comment le temps s'écoule dans cet univers ?

1. Calendrier

L'univers utilise le calendrier grégorien actuel.

Les dates, les mois et les années sont identiques à ceux du monde réel.

2. Jour / Nuit

Le cycle jour/nuit est entièrement naturel.

La luminosité influence directement :

les déplacements ;
la visibilité ;
la fatigue ;
le moral ;
certaines interactions ;
certains événements.

La nuit n'est pas forcément dangereuse, mais elle impose un rythme différent.

3. Saisons

Les quatre saisons existent.

Chaque saison modifie naturellement :

la température ;
la durée des journées ;
la météo ;
la végétation ;
les ressources disponibles ;
certaines difficultés de progression.
4. Passage du temps

Le temps s'écoule de façon continue.

Il ne s'arrête jamais pendant les dialogues ou les décisions.

Les journées, les nuits et les heures influencent naturellement l'évolution de l'aventure.

5. Vieillissement

Le vieillissement existe comme dans le monde réel.

Pour cette aventure, sa progression est négligeable en raison de la courte durée du récit.

6. Événements cycliques

Le monde suit ses propres cycles naturels :

lever et coucher du soleil ;
météo ;
saisons ;
comportement de la faune ;
croissance de la végétation.

Ces cycles continuent indépendamment des personnages.

7. Époque

L'aventure se déroule à notre époque.

Les références culturelles, les technologies et les habitudes de vie correspondent au monde contemporain.

Chapitre 5 — Ambiance

Question :

Quelle émotion dégage naturellement cet univers ?

1. Ambiance générale

L'univers dégage une sensation de réalisme, d'immersion et de solitude.

La nature est omniprésente, majestueuse, parfois rassurante, parfois hostile, mais jamais malveillante.

Le monde ne cherche ni à aider ni à punir les personnages : il existe simplement.

2. Climat émotionnel

L'univers oscille naturellement entre :

le calme ;
la contemplation ;
l'incertitude ;
la tension ;
l'espoir.

Les émotions naissent principalement des situations vécues plutôt que du décor lui-même.

3. Bande sonore / ASMR

L'univers privilégie les sons naturels.

Exemples :

le vent dans les arbres ;
les oiseaux ;
les insectes ;
le bruit des pas ;
le crépitement d'un feu ;
la pluie ;
une rivière ;
le tonnerre ;
les feuilles ;
le silence.

Il n'y a pas de musique de fond permanente.

La nature constitue l'ambiance sonore principale.

4. Météo dominante

La météo évolue de façon naturelle.

Alternance possible de :

soleil ;
nuages ;
pluie ;
brouillard ;
vent ;
orages ;
froid nocturne.

La météo influence directement les déplacements, le moral et les interactions.

5. Luminosité

La lumière suit le cycle naturel du soleil.

Les différentes périodes de la journée créent des atmosphères variées :

lever du jour ;
pleine journée ;
coucher du soleil ;
crépuscule ;
nuit.

Chaque ambiance lumineuse influence la perception de l'environnement.

6. Atmosphère

L'univers donne constamment l'impression d'être vivant.

Le joueur ressent :

l'immensité des paysages ;
l'isolement ;
la fragilité humaine ;
la beauté de la nature ;
le passage du temps.

Même dans les moments calmes, le monde continue d'exister autour des personnages.

7. Niveau de danger

Variable.

Le danger ne provient pas d'ennemis omniprésents mais des conditions naturelles :

fatigue ;
météo ;
relief ;
erreurs ;
manque de ressources.

Le monde n'est ni sûr ni dangereux par nature : ce sont les circonstances qui le rendent plus ou moins hostile.

Chapitre 6 — Événements dynamiques

Question :

Que peut produire naturellement le monde ?

1. Météo

Le monde génère naturellement des variations météorologiques.

Exemples :

pluie ;
vent ;
brouillard ;
orage ;
neige (selon la saison) ;
canicule ;
gel nocturne.

La météo influence directement les déplacements, la visibilité, les ressources et le comportement des personnages.

2. Catastrophes

Des phénomènes naturels peuvent survenir sans être liés au scénario.

Exemples :

éboulement ;
chute d'arbre ;
glissement de terrain ;
crue d'une rivière ;
incendie de forêt.

Ils modifient durablement l'environnement.

3. Guerres

Aucune influence directe.

Le monde contemporain peut connaître des conflits, mais ils ne concernent pas cette aventure.

4. Fêtes

Le monde possède ses propres événements sociaux (jours fériés, fêtes locales, etc.).

Ils existent mais restent hors du champ de cette aventure.

5. Migrations

La faune se déplace naturellement selon les saisons.

Certaines espèces peuvent apparaître ou disparaître selon la période de l'année.

6. Phénomènes naturels

L'univers produit continuellement des phénomènes réalistes :

lever et coucher du soleil ;
brouillard matinal ;
chute des feuilles ;
floraison ;
montée des eaux ;
gel ;
cycles des animaux.

Le monde évolue sans attendre les personnages.

7. Phénomènes surnaturels

Aucun.

Ils sont incompatibles avec le genre Réaliste.

8. Événements politiques

Les événements politiques du monde existent mais n'ont aucune incidence directe sur cette aventure.

9. Événements économiques

Le monde continue de fonctionner normalement en dehors de la zone du crash.

Ces événements n'influencent pas directement le récit.

10. Événements exceptionnels

L'univers peut produire des événements rares mais crédibles.

Exemples :

passage d'un avion de recherche ;
découverte d'une ancienne cabane ;
arrivée précoce de l'hiver ;
chute d'un arbre bloquant un passage ;
rencontre exceptionnelle avec un animal rare.

Ces événements restent réalistes et émergent naturellement des règles du monde.

Chapitre 7 — Interactions

Question :

Comment l'univers influence-t-il les personnages, le joueur et l'aventure ?

1. Influence sur les émotions

L'environnement agit naturellement sur les émotions des personnages.

Exemples :

une nuit froide accentue le stress ;
un lever de soleil redonne de l'espoir ;
plusieurs jours de pluie diminuent le moral ;
un paysage magnifique apaise les tensions ;
le silence d'une forêt favorise les confidences.

Le monde ne crée pas les émotions, il les influence.

2. Influence sur les décisions

L'univers oblige régulièrement les personnages à prendre des décisions.

Par exemple :

attendre ou continuer malgré la pluie ;
dormir maintenant ou marcher encore ;
traverser une rivière ou chercher un autre passage ;
économiser les ressources ou les utiliser immédiatement.

Les décisions naissent des contraintes du monde.

3. Influence sur les dialogues

Le contexte modifie naturellement les conversations.

Un même sujet n'est pas abordé de la même façon :

sous une tempête ;
autour d'un feu ;
après plusieurs jours de marche ;
en découvrant une trace de civilisation.

Le monde donne naissance aux dialogues.

4. Influence sur les relations

L'environnement crée des occasions de renforcer ou fragiliser la relation.

Par exemple :

coopérer pour franchir un obstacle ;
partager le dernier repas ;
traverser ensemble une nuit difficile ;
se soutenir après un échec.

Le monde rapproche ou éloigne les personnages sans jamais imposer le résultat.

5. Influence sur la progression

La progression dépend constamment de l'univers.

Le relief, la météo, les ressources et le temps peuvent :

ralentir ;
accélérer ;
détourner ;
compliquer ;
faciliter le voyage.

L'univers rythme naturellement l'aventure.

6. Influence sur les événements

Le monde est à l'origine de nombreux événements.

Exemples :

changement de météo ;
chute d'un arbre ;
montée des eaux ;
brouillard ;
passage d'animaux ;
découverte d'un refuge.

L'univers génère ces événements sans intervention du scénario.

7. Influence sur la mémoire

Le monde participe directement à la création des souvenirs.

Les personnages ne retiennent pas uniquement leurs conversations.

Ils se souviennent également :

d'un coucher de soleil ;
d'une nuit glaciale ;
du premier feu réussi ;
d'une rivière traversée ensemble ;
d'un refuge improvisé ;
d'un paysage exceptionnel.

Le décor devient lui aussi un élément de la mémoire.
